# AllCups_AvitoPrice. Задача с тренировочного этапа всероссийской олимпиады по ИИ 
# [Задача в соревновании](https://cups.online/ru/workareas/vserosii_2026/1553/3096)

Суть решения: 

Решение занимает первое место в лидерборде на момент 8 июля.

- Прогоняем все данные изображения через визуальный трансформер [DinoV2](https://huggingface.co/docs/transformers/model_doc/dinov2). 
- На получившихся эмбеддингах обучаем полносвязный слов на предсказание цены.
- Добавляем предсказания сети в табличные данные для catboost. Так-же в табличных данных мультивыборные фичи раскрываем в отдельные признаки
- Обучаем catboost, с помощью grid_search
